MathLibre-c is a project for building containers with mathematical software.

数学ソフトウェア環境のためのコンテナを開発するプロジェクト

# 必要な環境
Linux, Windows + WSL2, macOS + Homebrew

## Linux
### パッケージ podman のインストール
apt install podman もしくは yum install podman
